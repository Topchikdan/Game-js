# Кликер — Vanilla JS Game

**Браузерная кликер-игра с уровнями, прогресс-баром и случайными бонусами — без фреймворков, только чистый HTML, CSS и JavaScript.**

## ✨ Возможности

- 🖱 **Основная механика** — нажимай кнопку и набирай очки
- 📈 **Система уровней** — прогресс-бар заполняется с каждым кликом; при заполнении — новый уровень
- 🎲 **Случайные бонусы** — с каждым кликом есть шанс появления бонусной кнопки
- ⚡ **Мультиклик** — бонусы `+1` и `U` увеличивают количество очков за каждый клик
- 💥 **Четыре типа бонусов:**
  - `+1` — прибавляет +1 к мультиклику навсегда
  - `Q` — удваивает текущий счёт
  - `A` — мгновенно добавляет +5 к счёту
  - `U` — ульта: прибавляет +10 к мультиклику навсегда
- ⏳ **Таймаут бонусов** — бонусная кнопка исчезает через 8 секунд, если не нажать

## 🎮 Механика бонусов

Бонусы выпадают случайно при каждом клике. Генерируется случайное число от 0 до 100 — если оно равно 0, 1, 2 или 3, появляется соответствующий бонус (шанс ~4% на каждый тип). Бонусы добавляются в панель вверху и исчезают после нажатия или через 8 секунд.

## 🛠 Технологии

- **HTML5** — семантическая разметка
- **CSS3** — стили разбиты по модулям через `@import`
- **Vanilla JavaScript ES6+** — модульная система (`import` / `export`), без библиотек

## 📁 Структура проекта

```
clicker/
├── index.html          # Разметка страницы
├── index.css           # Точка входа для стилей (импортирует модули)
├── index.js            # Основная логика: клик, уровни, прогресс-бар
├── bonus.js            # Логика генерации и обработки бонусов
├── config.js           # Глобальное состояние и DOM-элементы
└── assets/
    ├── global.css      # Базовые стили и сброс
    ├── clicker.css     # Стили основного блока кликера
    └── clicker-bonus.css  # Стили бонусных кнопок
```

## 🚀 Как запустить

Сборка не нужна, но файл открывается **только через локальный сервер** из-за ES-модулей (`type="module"`):

# Клонировать репозиторий
git clone https://github.com/Topchikdan/Game-js.git

# Перейти в папку
cd clicker

# Запустить локальный сервер (через Live Server в VS Code):

> ⚠️ Просто открыть `index.html` двойным кликом **не сработает** — браузер заблокирует ES-модули по политике CORS. Нужен любой локальный HTTP-сервер.

## 📌 Примечания

- Состояние игры **не сохраняется** при перезагрузке страницы (нет localStorage)
- CSS-файлы в папке `assets/` нужно создать самостоятельно или добавить свои стили для классов: `.clicker`, `.clicker__button`, `.clicker__score`, `.clicker__progress`, `.clicker__bunus-button`, `.clicker__bunus-button--0/1/2/3`
