## О проекте

Браузерная кликер-игра на Vanilla JavaScript (ES6-модули). Нажимай кнопку, набирай очки, повышай уровень и собирай случайные бонусы. Сборка не требуется, но запуск через локальный сервер обязателен из-за ES-модулей.

---

## Возможности

- Основная механика: нажимай кнопку и набирай очки
- Система уровней: прогресс-бар заполняется с каждым кликом, при заполнении — новый уровень
- Случайные бонусы: с каждым кликом есть шанс появления бонусной кнопки
- Мультиклик: бонусы `+1` и `U` увеличивают количество очков за каждый клик навсегда
- Четыре типа бонусов: `+1` (+1 к мультиклику), `Q` (удвоить счёт), `A` (+5 к счёту), `U` (+10 к мультиклику)
- Таймаут бонусов: бонусная кнопка исчезает через 8 секунд, если не нажать

---

## Механика бонусов

При каждом клике генерируется случайное число от 0 до 100. Если оно равно 0, 1, 2 или 3 — появляется соответствующий бонус (шанс ~4% на каждый тип). Бонусы добавляются в панель вверху и исчезают после нажатия или через 8 секунд.

## Быстрый старт

> Просто открыть `index.html` двойным кликом не сработает — браузер заблокирует ES-модули по политике CORS. Нужен локальный HTTP-сервер.

```bash
git clone https://github.com/Topchikdan/Game-js.git
cd Game-js
npx serve .
```

Или откройте через Live Server в VS Code.

## Структура проекта

```
clicker/
├── index.html             # разметка страницы
├── index.css              # точка входа для стилей (импортирует модули)
├── index.js               # основная логика: клик, уровни, прогресс-бар
├── bonus.js               # логика генерации и обработки бонусов
├── config.js              # глобальное состояние и DOM-элементы
└── assets/
    ├── global.css         # базовые стили и сброс
    ├── clicker.css        # стили основного блока кликера
    └── clicker-bonus.css  # стили бонусных кнопок
```

## Примечания

- Состояние игры не сохраняется при перезагрузке страницы (localStorage не используется)
- CSS-файлы в папке `assets/` нужно создать самостоятельно или добавить свои стили для классов: `.clicker`, `.clicker__button`, `.clicker__score`, `.clicker__progress`, `.clicker__bunus-button`, `.clicker__bunus-button--0/1/2/3`
